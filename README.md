# dotFiles

## macOS

### install homebrew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

### install configuration `install.sh`

   ```
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/alfredotang/dotfiles/main/macOS/install.sh)"
   ```

### install with symlinks `symlink.sh`

   ```
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/alfredotang/dotfiles/main/macOS/install_with_symlink.sh)"
   ```
